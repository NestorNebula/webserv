WEBSERV

Demonstration Website

for our HTTP server

The Project

Webserv is an HTTP server written in C++98.

Like any HTTP server, it can handle multiple connections at the same time and provides an accurate response to each request received.

The server accepts the GET, POST and DELETE methods.

It can handle file uploads and execute CGIs.

This project also includes a demonstration static website (on which you may be reading this now!) to demonstrate some of its functionalities.
